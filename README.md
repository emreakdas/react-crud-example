
## VerdExport Case

Bu proje VerdExport şirketi için yapılan case çalışmasıdır.

Canlı URL: https://verdexport-case.vercel.app/


## Başlayın 

#### Repoyu klonlayın

```bash 
  git clone https://github.com/emreakdas/verdexport-case.git
```
    

#### Dizine gidin

```bash 
  cd verdexport-case
```

#### Bağımlılıkları yükleyin

```bash 
  pnpm install
```

#### Projeyi kaldırın

```bash 
  pnpm run dev
```
